## Acknowledgements

Cognito would not exist without the prior art it builds on:

- **[Sinapsis](https://github.com/Luispitik/sinapsis)** — deterministic-hooks pattern and continuous-learning architecture.
- **Edward de Bono** — *Six Thinking Hats* (1985), the foundational influence on the seven-mode model.
- **Daniel Kahneman** — *Thinking, Fast and Slow* (2011), source of the System 1 / System 2 framing used to justify deterministic gates.
- **Charlie Munger, Genrich Altshuller (TRIZ), Bob Eberle (SCAMPER)** — mental-model catalogue feeding the Divergent mode.
- **The Anthropic team** — Claude Code, the host runtime that makes hooks, skills, and slash commands possible.

## Communities

- **[IA Masters Automation](https://www.skool.com/ia-masters-automations/about?ref=b80c70e7e20543c69a9ee1044dab14c5)** — Spanish-speaking community on AI automation, Claude Code patterns, and operator tooling. Many of the patterns and feedback loops that shaped Cognito were workshopped here.
